## General Sibling Selector
General Sibling Selectors General sibling selectors apply to any subsequent sibling of a particular element.
```html
<h1> Transport </h1>*
<p> Take bus 11 towards Pengam Green. </p>
<p> Alternatively, just walk. </p>
```
```css
h1~p {color: red; }
```
