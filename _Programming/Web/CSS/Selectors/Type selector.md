# Selectors
## Type selectors
Type selectors apply to a particular type of element (e.g. h1, p or em).
```html
<h1> Main Attractions </h1>
<p> <em>Splott Beach</em> and
	<em>Splott Market</em> are the main attractions in Splott
</p>
```
```css
h1,p { color: red; }
em { color: blue; }
```
